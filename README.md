# SunDiary🌞🌛

## About this app
This app is a diary app.    
But, you are not a human. You're the Sun. so, you must write diaries becoming the Sun.  
For example, you can write diary about the Earth(watched by the sun), the Universe.

### functions
- You can get diary's advice by the Moon(AI) (but, it didn't finished. definitely add someday).
- This app has a Star Screen. There are some star(random color and random offset). Amount of stars is your written diaries numbers.
- You can search every diary by word. (you can also select Edited, NotEdited, IsLiked... diary)
- Your diary can have a picture.
